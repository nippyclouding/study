# 카카오 지도 API 연동 가이드

## 1. 시작하기 (애플리케이션 등록 및 API 키 발급)
1. **Kakao Developers (카카오 개발자 센터)**에 로그인합니다.
2. 내 애플리케이션을 생성하고 클릭하여 들어갑니다.
3. 왼쪽 메뉴에서 **[앱 설정] -> [플랫폼 키]**를 클릭합니다.
4. **[JavaScript 키]** 항목을 클릭하여 상세 페이지로 들어갑니다.
5. **[JavaScript SDK 도메인]** 항목에 `http://localhost:5500` (또는 라이브 서버 주소)이나 `http://127.0.0.1`을 등록합니다. (이 과정이 없으면 지도가 뜨지 않습니다.)
6. 해당 **JavaScript 키** 값을 복사해 둡니다.

## 2. 문서에서 '시작하기' 예제 코드 가져오기
1. 카카오 지도 API 공식 문서의 **지도 띄우기 가이드**로 이동합니다. https://developers.kakao.com/docs/ko/kakaomap/common
2. 화면에 있는 HTML/JS 코드를 복사해서 작업 중인 HTML 파일(`kakao-map-api.html`)에 붙여넣기 합니다.

## 3. 내 API 키 적용하기
1. 복사한 코드 중 `<script type="text/javascript" src="//dapi.kakao.com/v2/maps/sdk.js?appkey=발급받은_APP_KEY"></script>` 부분에 방금 복사했던 **내 JavaScript 키**를 붙여넣습니다.

## 4. 결과 확인 및 응용
1. 브라우저를 열어 지도가 잘 뜨는지 확인합니다.
2. 지도가 잘 뜬다면 공식 문서의 **Sample 메뉴**로 이동해 봅니다.
3. '마커 생성하기', '인포윈도우 띄우기', '클릭 이벤트 등록하기' 등의 예제 코드를 내 지도에 하나씩 추가해 보면서 문서의 규격을 이해해 봅니다.
